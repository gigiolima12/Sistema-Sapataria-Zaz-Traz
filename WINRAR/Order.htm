Como comprar uma licença do WinRAR e RAR.

Se você quiser continuar usando o WinRAR e o RAR após o período de avaliação de 40 dias, você precisa comprar uma licença de um dos representantes regionais listados aqui. O tradutor oferece este link seguro para compras no Brasil. Você pode também verificar a lista mais recente de preços e registrar-se online em www.win-rar.com.

Após o recebimento de sua taxa de registro você receberá um e-mail contendo um arquivo de chave de registro correspondente ao nome do usuário e a string que você escolheu. Por favor especifique um endereço de e-mail válido ao comprar a licença, pois é para este endereço que será enviada a chave de registro.

O e-mail de registro conterá também todas as instruções necessárias, então por favor siga os passos. Abaixo, nós fornecemos uma breve explicação do procedimento típico de registro, porém as instruções que estarão no e-mail são mais atuais que as contidas neste arquivo, deste modo deve ter uma precedência maior.

Se você usa o WinRAR, você precisa copiar o arquivo de registro (rarreg.key) para a pasta do WinRAR ou para a pasta %APPDATA%\WinRAR. Por padrão esta pasta é "C:\Program Files\WinRAR" (em Inglês) ou "C:\Arquivos de Programas\WinRAR" (em português), mas pode ser alterada pelo usuário no momento da instalação do WinRAR. Você pode também arrastar o arquivo rarreg.key e soltá-lo na janela do WinRAR para registrar.

Se a chave de registro estiver comprimida em um arquivo .rar ou .zip, você deve extrair o arquivo rarreg.key antes de copiá-lo. Se o nome do arquivo é rarkey.rar, uma outra maneira para instalar o arquivo da chave é abrir esse arquivo no WinRAR e responder "Sim" para solicitação de confirmação.

Se você usa o RAR/Unix e RAR para macOS, você deverá copiar o arquivo rarreg.key para seu diretório principal ou para um dos seguintes diretórios: /etc, /usr/lib, /usr/local/lib, /usr/local/etc. Você deve renomeá-lo para .rarreg.key ou .rarregkey, se você quiser, mas rarreg.key também é válido.

O WinRAR, linha de comando do RAR para Unix e macOS agora usam o mesmo formato de chave de registro, assim você pode utilizar a mesma chave com as atuais versões WinRAR e RAR para todas as plataformas mencionadas. Não é garantido para versões WinRAR e RAR que não são iguais as versões incluídas nesta distribuição.
